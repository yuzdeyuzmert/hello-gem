# Hello

A simple Hello World Gem, written by Mert AKSOY

## Installation

Add this line to your application's Gemfile:

    gem 'hello'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hello

## Usage

% irb
>> require 'hello'
=> true
>> HEllo.hi
Hello world!
